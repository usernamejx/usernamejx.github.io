默认情况下win11的触控板操作

点击

  • 单指点击 : 鼠标左键功能

    单指移动光标，单指双击=鼠标左键双击，单指双击（不抬起）拖动文件=单指重按（不抬起）拖动文件=鼠标左键按下（不抬起）拖动文件

  • 双指点击 : 鼠标右键功能

  • 三指点击 : 搜索框页面, 并定位搜索

  • 四指点击 : 消息中心

滑动

滑动 : 将手指放在触控板上, 不要让手指离开触控板, 摩擦移动

  • 单指 : 移动光标你学会了吗?🤣

  • 双指 :

    1. 双指上下滑动移动屏幕
    2. 看图时双指左右滑动切换图片
    3. 双指往外滑动放大往内滑动缩小当前内容

  • 三指 :

    1. 下, 三指同时向下滑动, 返回桌面

    2. 上, ( 上一步为三指同时向下滑动时 ) 三指同时向上划, 返回刚才的页面

      即撤销, 返回上一个窗口

    3. 上, ( 否则 ) 三指同时向上划, 显示所有桌面和当前桌面的页面

    4. 左, 三指向左滑动, 不要让手指离开触控板, 不要让手指离开触控板, 不要让手指离开触控板, 慢慢移动, 然后看到蓝色的框框在动, 慢慢移动到该去的地方

    5. 右, 三指向左滑动, 不要让手指离开触控板, 不要让手指离开触控板, 不要让手指离开触控板, 慢慢移动, 然后看到蓝色的框框在动, 慢慢移动到该去的地方

  • 四指 :

    1. 左右滑动, 改变桌面 ( 如果你有建立一个的话 )

改变默认设置

开始 —> 设置 —> 蓝牙和其他设备 —> 触摸板
